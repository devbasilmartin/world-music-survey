# World Music Survey — build workspace

Dedicated repo for continuing the deep, chronological, listenable survey of the
world's music, region by region. The United States and Europe are complete;
Latin America is in progress. Control docs live at the repo root; per-region
installments and masters are generated by `build_*.py` scripts.

## The loop (one iteration = one installment)

Driven by `CLAUDE_CODE_CONTINUE_SURVEY.md`. Each iteration:

1. Open the status tracker in `WORLD_MUSIC_SURVEY_ROADMAP.md`; find the first
   region that is IN PROGRESS, else the first "not started" in build order.
2. Determine the next installment for that region (era template / planned list).
3. Build ONE installment via a `build_<name>.py` script that holds the data as
   `(section_title, intro, [(title, artist_or_None), ...])` and emits both:
   - `<region>_<N>_<ERA>.md` — narrative (title + framing para, 5–10 sections,
     each an intro + `- Title — Artist` / `- Title — (Traditional)` list).
   - `<region>_<N>_<ERA>_IMPORT.txt` — clean `Title - Artist`, one per line;
     bare title for traditional pieces with no definitive recording.
4. Update the roadmap status table + append a Progress-log line.
5. On a region's last installment, also emit `MASTER_GUIDE_<region>_survey.md`
   and the deduplicated `MASTER_<REGION>_FULL_SURVEY_IMPORT.txt`
   (dedup key = lowercased title+artist, parentheticals & punctuation stripped,
   keep first occurrence in chronological file order).
6. Commit the installment; the scheduled trigger fires the next iteration.

Target ~30–90 tracks/installment, ~6–12 per section. Weave art music + folk +
popular chronologically.

## Hard rules (repeatedly caused errors — do not repeat)

- ONE song per line. Never cram two titles into one entry.
- Region integrity: verify every artist belongs to the region; flag judgment
  calls with a stated reason in the `.md`.
- Preserve difficult history with context + content warnings; don't sanitize.
- Cross-link connected traditions in the intro prose.
- Dedup per region master as described above.

## Build order

Latin America (in progress) → Sub-Saharan Africa [West, Southern, Central,
East, Sahel] → India/South Asia → East Asia [China, Japan, Korea] → MENA →
Southeast Asia → Caribbean → Central Asia & Caucasus → Oceania/Pacific →
Jewish diaspora → (optional global-fusion capstone + grand master index).
